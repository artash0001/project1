# Ideal Workday (Realistic)

## Morning
Goal: wake up, not win the day.

- Wake up
- Light routine (movement / shower / coffee)
- No decisions
- No work planning

---

## Focus Block
Goal: ONE finished thing.

- Choose task using Daily Start Protocol
- Define finish line
- Work until done or energy drops
- Save result (file / commit / message)

---

## Reset
Goal: clear the head.

- Short break
- No screens if possible
- Physical reset

---

## Maintenance Block
Goal: reduce background stress.

- Messages
- Payments
- Admin
- Small tasks

---

## Evening
Goal: stop the day cleanly.

- No new tasks
- Light review:
  - What did I finish?
  - Was today a good day?
- Shut down work mentally
