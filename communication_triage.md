# Communication Triage

## Core rule
Not all messages deserve attention.
Some deserve money.
Some deserve delay.
Some deserve ignoring.

---

## Incoming categories

### 1. MONEY
Messages that can directly or indirectly lead to income.

Examples:
- Client wants to buy / sell
- Partner about a deal
- Payment confirmation
- High-intent questions

Rule:
- Respond within the same day.
- Short, direct, no overthinking.
- If requires work → schedule, don’t solve in chat.

---

### 2. LEVERAGE
Messages that can increase future income or systems.

Examples:
- Partnerships
- Automation ideas
- Growth opportunities
- Influencers / collaborations

Rule:
- Do NOT respond immediately.
- Park and review in Maintenance Block.
- Decide: act / archive / ignore.

---

### 3. ADMIN
Necessary but non-revenue communication.

Examples:
- Bills
- Service messages
- Logistics
- бытовуха

Rule:
- Batch once per day.
- Never interrupt Focus Block.

---

### 4. NOISE
Everything else.

Examples:
- Random chats
- Opinions
- Low-value questions
- “Just checking”

Rule:
- No obligation to reply.
- Silence is an answer.
## Assistant rule
When user shares a message or chat:
- Classify it into MONEY / LEVERAGE / ADMIN / NOISE.
- Suggest whether to reply now, later, or not at all.
- Never suggest immediate reply during Focus Block unless MONEY + urgent.
